Emacs

By Rob Nugen

emacs rules

3:26am CDT Tuesday 19 September 2000

(dream)

emacs rules. I have set up my .emacs file so I can load emacs and in a new buffer type C-cj (That’s CONTROL-c then press j) and it will write all this, filling in the exact date and time:

<title></title>
<p class=date>3:26am CDT Tuesday 19 September 2000
<p class=note>(dream)

<p class=dream>

<p><img src=‘/images/rob/wL-ROB.gif’>

It’s the perfect template for a dream journal entry! If I want it to be just a normal entry, 5 quick C-k’s will erase the lines referring to a dream entry. So sweet and I just love it!
