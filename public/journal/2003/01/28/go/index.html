Yahoo! It works. I'm impressed that it does work. Janette's computer is configured the same way it has been - to use PPPoE to connect to her DSL provider with username and password. I don't know how the hub is so smart to let one computer's username and password....... hmm I have an idea. Maybe the DSL provider just gets a username and password from anywhere and then lets the rest of the data traffic flow, no matter where it wants to go. Sounds good to me.
