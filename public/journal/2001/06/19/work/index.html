Work

By Rob Nugen

4:44pm Tuesday 19 June 2001

Work has been pretty okay; I cleared a lot of crap off my desk, and have been on a conference call with one of our parnters about a new client that will be using the G3.

This is all great and good, but it really seems to slow me down in the overall production of G3 because I don't have the tools built quite yet that will make turning up a new project super duper simple.

It's pretty darn easy, but not simple. Definitely not trivial.

And I can't make it trivial if I have to keep setting up new projects all the time.

But don't get me wrong; I'm *glad* the G3 is being well utilized. I'm excited that everyone loves it. I just want more time to hang it within a nice administrative framework so we can easily manipulate shit in the thing.

I think I know one thing that I can write that will make manipulation significantly easier. I'm not sure how to write a generalized version though. Hmmm.
