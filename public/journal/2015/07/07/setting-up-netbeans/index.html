At comveni after solving the puzzle I couldn’t solve last week at the toy store! Photos were taken to commemerate the moment.

  1. Clone the source code locally
  2. Create new project
  3. Choose PHP
  4. Choose PHP Application with Existing Source
  5. Click Next
  6. Cut-n-paste path to repo in Sources Folder
  7. Confirm Put NetBeans metadata… is selected
  8. Click next
  9. Configure Run As Remote Web Site
  10. Create entry in /etc/hosts file
  11. Copy /etc/hosts url into Project URL
  12. Create new Remote Connection (if necessary)
  13. Choose existing Remote Connection (otherwise)
  14. Choose “On Save” for Upload Files
  15. Check that the Upload Directory matches that on virtual box
  16. Click Finish
