By Rob Nugen

##15:56 Wednesday 17 December 2014 JST##

Sweet this Limited Express train is fun to ride as it whizzes past like 7 stations between KQ Kawasaki and Yokohama (sorry, Travis). I might try to ride in the very front (passenger) seat next time. It features big windows to see out front.

To be honest, I’ve never really liked riding trains from KQ Kawasaki; there are several lines which use the same set of platforms so I can’t just jump on any train that comes by.
