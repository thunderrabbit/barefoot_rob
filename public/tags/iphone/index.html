Get a big voice memo off my iPhone

By Rob Nugen

##11:11 Monday 24 November 2014 JST##

Visit https://www.macroplant.com/iexplorer/

Click Download

Open the .dmg file

Click, drag, and drop the left icon onto the right one.

Click continue with demo

Click voice memos

Select the memo(s) you want to download

click [Export seleced item(s) to folder…]

Download the file somewhere

The file will have a green checkmark once it has been transferred.
