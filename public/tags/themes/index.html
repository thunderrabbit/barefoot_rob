evaluating several hugo themes

By Rob Nugen

15:19 Tuesday 25 June 2019 JST

I have gone through all the themes in the repo for All Hugo’s Themes and come up with a list of several which do a lot of these things:

  • content on top page
  • list page for blog/
  • list page for journal/
  • paginate blog/
  • paginate journal/
  • show tags
  • paginate tags
  • show categories
  • paginate categories
  • nicely show Youtube videos

I don’t know if a single one does all the above things, but here is my list so far:
