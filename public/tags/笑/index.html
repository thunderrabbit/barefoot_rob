tomato harvest

By Rob Nugen

12:07 Wednesday 23 May 2018 JST

コンプリスにタイマーがあります。　There is a timer on Complice.

タイマーが終わったら、トマトをもらいます。　Every time the timer finishes, I get a tomato.

今日、タイマーの時間を０秒に設定しました。 Today I set the timer for 0 seconds.

０秒で、５千個以上トマトをもらった。 I got over 5000 tomatos.

猫起き

昨日の日本語の授業の時、うちの猫の座るのは面白くてよく笑った。

まず、猫はUSBのキーボードに座ったから私がキーボードのケーブルを抜いて 日本語喋りながら、猫を撫でた。

「ここはすごく邪魔じゃないんだね」と思って、猫はメインのキーボードに登って座った。

私と先生はよく笑ったから、猫がもううるさい！と思って、キーボードから出た。

猫が猫プロテクターの上でよく寝た。
